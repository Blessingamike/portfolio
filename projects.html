My Personal Projects

Word Guessing Game

The game presents the user with two words to guess their connection within five attempts. Each wrong guess reveals two additional hint words but reduces the score. The game ends when the user guesses correctly or exhausts all attempts, displaying their final score. Additionally, users can create custom mini-games by saving word connections and answers in a file via file I/O, which can later be used to play.
